DQL（Data Query Language）

数据查询语言，用来查询数据。

一、基本查询

1. 字段（列）查询

1）查询所有列
  • SELECT * FROM 表名；
2）查询指定列
  • SELECT 列名1 [, 列名2, …] FROM 表名；
3）查询不重复记录
  • 当查询结果中有多行记录一模一样时，只显示一行。一般在只查询一列或数列时会出现这种情况；
  • SELECT DISTINCT * | 列名1 [, 列名2, …] FROM 表名；
4) 列运算

  • I 数量类型的列可以做加、减、乘、除运算：

    • SELECT sal*1.5 FROM emp ;
    • SELECT sal+comm FROM emp ;

  • II 字符串类型可以做连续运算：

    • SELECT CONCAT(‘$’, sal) FROM emp ;
    • 只能使用CONCAT来连接字符串；

  • III 转换NULL值：

    • SELECT IFNULL(comm, 0) FROM emp ;
    • 把表记录中的NULL替换；

  • IV 给列起别名：

    • SELECT 列名 [[AS] 新列名] FROM 表名 ；
    • 可以使用AS来更改显示时的列名，而且AS可以省略，注意中间没有逗号；

2. 条件控制

1）条件查询
  • 使用WHERE语句来实现条件查询；
  • SELECT empno, sal WHERE sal BETWEEN 20000 AND 40000 ;
  • SELECT ename FROM emp WHERE job IN (‘经理’, ‘董事长’) ;
  • SELECT ename FROM emp WHERE job IS NOT NULL ;

2) 模糊查询

  • 例：想查询姓张，而且姓名一共有两个字的员工时，可以使用模糊查询；

  • SELECT * FROM emp WHERE ename LIKE ‘张_’ ;

    • 使用LIKE运算符来匹配字符，注意这里只能匹配一个字符；
    • 上面的查询只能匹配名字为两个字，而且是姓“张”的员工；

  • SELECT * FROM emp WHERE ename LIKE ‘___’ ;

    • 这里匹配的就是名字为三个字的员工；

  • 如果查询姓“张”而且名字数目不限的员工就可以使用”%“；

    • SELECT * FROM emp WHERE ename LIKE ‘张%’ ;
      • ‘%‘表示匹配0~N个任意字符串；
    • SELECT * FROM emp WHERE ename LIKE ‘%啊%’ ;
      • 匹配名字中带有“啊”字的员工；
    • SELECT * FROM emp WHERE ename LIKE ‘%’ ;
      • 这个条件等同于不存在，但是姓名为NULL的就匹配不出来；

二、排序

1. 升序

  • SELECT * FROM emp ORDER BY sal ASC ;
    • ASC表示按照sal升序排列，而且默认是升序排列，因此ASC可以省略但不建议；

2. 降序

  • SELECT * FROM emp ORDER BY comm DESC ;
    • DESC表示按照comm降序排列；

3. 使用多列作为排序条件

  • SELECT * FROM emp ORDER BY sal ASC comm DESC enpno ASC ;
    • 从前往后一次满足排序条件，如果记录值相等则匹配后面的条件；

三、聚合函数

  • 聚合函数用来做某列的纵向运算；

1. COUNT

  • SELECT COUNT(*) FROM emp ;
    • 计算表中所有列中都不为NULL的记录的行数；
    • SELECT COUNT(100) FROM emp ;
    • SELECT COUNT(ename) FROM emp ;
    • 括号里用*和数字的结果是一样的；

2. MAX

  • SELECT MAX(sal) FROM emp ;
  • 最大值

3. MIN

  • SELECT MIN(sal) FROM emp ;
  • 最小值

4. AVG

  • SELECT AVG(sal) FROM emp ;
  • 平均值

5. SUM

  • SELECT SUM(sal) FROM emp ;
  • 求和

四、分组查询

1. 分组查询是把记录使用某一列进行分组，然后查询组信息；

  • 例：查看所有部门的记录数
    • SELECT deptno, COUNT(*) FROM emp GROUP BY deptno ;
      • 使用deptno分组，查询部门编号和每个部门的记录数；
    • SELECT job MAX(sal) FROM emp GROUP BY job ;
      • 使用job分组，查询每种工作的最高工资；
2. 分组条件查询
  • SELECT deptno, COUNT(*) FROM emp WHERE sal > 10000 GROUP BY deptno HAVING COUNT(*) > 3 ORDER BY deptno ASC ;
  • 以部门分组，查询每组sal大于10000的记录数，且条件为记录数大于3，并按照部门顺序排列；
  • 顺序是依次为SELECT -> FROM -> WHERE -> GROUP BY -> HAVING -> ORDER BY ;

五、limit子句（方言）

(只有mysql里面才有limit)

  • LIMIT用来限定查询结果的起始行，以及总行数；
  • 例：查询起始行为第5行，一共查询3行记录；
    • SELECT * FROM emp LIMIT 4, 3 ;
    • 第1行的下标为0；
