多表查询

分类

（一）合并结果集（了解）

  • 要求被合并的表中，列的类型和列数相同
  • UNION：去重
  • UNION ALL：不去重
create table ab (a int, b varchar(50));
insert into ab values (1, '1');
insert into ab values (2, '2');
insert into ab values (3, '3');

create table cd (c int, d varchar(50));
insert into cd values (3, '3');
insert into cd values (4, '4');
insert into cd values (5, '5');

select * from ab
union [all]
select * from cd ;

（二）连接查询

1. 内连接

  • 需要去除笛卡尔积：表1和表2种所有的记录会两两互相组合，需要条件筛选所有结果；
（1）方言
select * from 表1 别名1, 表2 别名2 where 别名1.xx=别名2.xx;
（2）标准
  • 表1 inner join 表2 on 条件
select * from 表1 别名1 inner join 表2 别名2 on 别名1.xx=别名2.xx;

select e.name e.deptno d.name
from emp e inner join dept d
on e.deptno=d.deptno;
（3）自然（自动匹配）
  • 表1 natural join 表2
select * from 表1 别名1 natural join 表2 别名2;

2. 外连接

  • join、left outer join、right outer join
  • 全外：full outer join（mysql不支持）
（1）左外：left outer join
select * from 表1 别名1 left outer join 表2 别名2 on 别名1.xx=别名2.xx;
-- 左外以左边的表为主，右边的表为辅
-- 左表记录无论是否满足条件都会被查询出来，而右表只有当满足条件时才显示
-- 左表中不满足条件的记录会将右表中的值设置为null
  • 左外自然：natural left outer join
（2）右外：right outer join
select * from 表1 别名1 right outer join 表2 别名2 on 别名1.xx=别名2.xx;
-- 右外以右边的表为主，左表为辅
  • 右外自然：natural right outer join
（3）全外：full outer join（mysql不支持）
  • 可以用union来实现
select * from 表1 别名1 left outer join 表2 别名2 on 别名1.xx=别名2.xx
union
select * from 表1 别名1 right outer join 表2 别名2 on 别名1.xx=别名2.xx;
-- 使用union将左外结果和右外结果做并集就能实现全外的效果

（三）子查询

1. 出现位置

  • 子查询可以在select后面（不标准）
  • 也可以在from后面作为表
  • 也可以在where后面作为条件（多行多列）
-- 查询本公司*工资最高的*员工的详细信息
select * from emp where sal = (select max(sal) from emp) ;

-- 查询30部门中员工编号和名字
select e.empno, e.ename from (select * from emp where deptno=30) e ;

2. 条件

（1）单行单列
select * from 表1 别名1 where 列1 [=, >, <, >=, <=, !=] (select 列 from 表2 别名2 where 条件) ;
（2）多行单列
select * from 表1 别名1 where 列1 [in, all, any] (select 列 from 表2 别名2 where 条件) ;
（3）单行多列
select * from 表1 别名1 where (列1, 列2) in (select 列1, 列2 from 表2 别名2 where 条件) ;
（4）多行多列
select * from 表1 别名1, (select ....) 别名2 where 条件 ;
